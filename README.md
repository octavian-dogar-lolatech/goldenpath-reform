# GoldenPath
GoldenPath wants to be an universal startup project for any kind of Java Rest API using SpringBoot and include features like Logging, Monitoring, Security, Authentification, etc.

## How to run

1.```git clone project``` 

2.```cd project_path```

3.```mvn clean install```

4.```mvn spring-boot:run```
